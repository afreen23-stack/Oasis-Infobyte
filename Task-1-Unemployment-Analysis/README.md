# Unemployment Analysis with Python

## Objective

The objective of this project is to analyze unemployment trends in India using Python.
This analysis focuses on understanding how the unemployment rate changed over time, especially during the Covid-19 lockdown period.

## Dataset

The dataset contains unemployment statistics for different states in India.
Main columns in the dataset:

* **Region** – State name in India
* **Date** – Date when the unemployment rate was recorded
* **Frequency** – Measurement frequency (Monthly)
* **Estimated Unemployment Rate (%)** – Percentage of unemployed people
* **Estimated Employed** – Number of employed people
* **Estimated Labour Participation Rate (%)** – Percentage of people participating in the labor force
* **Area** – Urban or Rural classification

## Steps Performed

1. Imported required Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.
2. Loaded and explored the unemployment dataset.
3. Cleaned the dataset by removing missing values and formatting columns.
4. Converted the Date column into datetime format.
5. Performed exploratory data analysis (EDA) using visualizations.

## Visualizations

The following visualizations were created to analyze unemployment trends:

* **Unemployment Rate by State**
* **Unemployment Rate Trend Over Time**
* **Correlation Matrix of Employment Factors**
* **Comparison of Rural vs Urban Unemployment Rates**

## Key Insights

* The unemployment rate increased sharply during the Covid-19 lockdown period.
* Certain states showed higher unemployment rates compared to others.
* Urban areas experienced slightly higher unemployment compared to rural areas.
* Employment levels have a negative relationship with unemployment rates.

## Tools and Technologies

* Python

## Conclusion

The analysis highlights how the Covid-19 pandemic significantly impacted employment opportunities in India. Visualizations help in understanding unemployment patterns across states and over time, providing insights into labor market trends.
